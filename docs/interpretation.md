# Model: From Biological Tension to Narrative Civilization

## 1. 物理底座 (The Physical Base)

**状态：** 客观差异 (Objective Differences)

**定义：** 环境、身体、结构之间的差异客观存在，但尚未构成“问题”。

- **Input:** 差异 (Difference)
- **Mechanism:** 激素/神经信号 (Hormonal/Neural Signals)
- **Result:** 系统状态改变 (System State Change) -> 生理/神经层偏离

## 2. 张力梯度 (Tension Gradient, TG)

**核心特征：** 纯生理、无意识、非叙事 (Non-narrative)。

- **本质：** 稳态偏离 (Homeostatic Deviation)
- **表现：** 饥饿、疼痛、疲劳、威胁

### 分岔路径 (Bifurcation)

- **路径 A [未被感知]:** 自动调节 (Automatic Regulation) -> 直接行动 -> **无体验/无情绪** (Pure system correction)。
- **路径 B [被感知]:** 体验出现 (Perceived) -> **前叙事感受 (Pre-narrative Affect)**。
    - 特征：无对象、无意义、无“我” (No object, No meaning, No self)。
    - 状态：紧/松/压/不适。

## 3. 关键转折：调节失效与认知冗余

当 **路径 B** 中的调节失败 (Regulation Fails)，即直接行动无法消除张力，且大脑具备 **认知冗余 (Cognitive Surplus)** 时：

1. **自反性 (Reflexivity):** 系统开始将自身状态作为对象进行观察。
2. **位势差 (Potential Gradient, PG):** 观察导致了“当前状态”与“评估状态”的结构性不对称。
    - *“现在” (Now) ≠ “应然/可能” (Could/Should)*

### PG 的多维空间 (Axis Space)

- **EPG (Evaluative):** 现在 vs 更好的自我
- **SPG (Social):** 自我 vs 他人/位置
- **TPG (Temporal):** 现在 vs 未来

## 4. 叙事接口 (The Narrativity Interface)

**问题 (The Problem):** 结构性处理瓶颈 (Structural Processing Bottleneck)。人脑无法并行消解多维的 PG。
**解决方案 (The Solution):** **叙事压缩 (Narrative Compression)**。

### 压缩过程 (The Process)

1. **视角绑定 (Perspective Binding):** 确定“我”。
2. **主轴选择 (Axis Selection):** 选择 EPG/SPG/TPG 中的一个作为主矛盾。
3. **因果简化 (Causal Simplification):** 建立归因链条。
4. **时间线性化 (Temporal Linearization):** 赋予时间顺序。

### 🚩 涌现时刻 (Emergence Point)

> EMOTION EMERGES HERE (情绪在此生成)
> 
> 
> 情绪 = 被归属、被命名、被解释的感受 (Affect becomes owned, named, and reasoned)。
> *情绪不是原始输入，而是叙事接口的**输出产物**。*
> 

## 5. 输出与外化 (Output & Externalization)

1. **意图生成 (Intent Formation):**
    - 定向行动方案 (Directional action schema)。
    - 语言化：我要 / 我应该 / 我必须。
2. **符号外显 (Symbolic Externalization):**
    - 形成可复制的叙事压缩包 (Language)。
3. **叙事对齐测试 (Narrative Alignment Test):**
    - **对齐 (Alignment):** -> 协作 / 文明 (Coordination / Civilization)
    - **失配 (Misalignment):** -> 焦虑 (Anxiety) *注：焦虑本质上是叙事失败，导致原始位势差(PG)的裸露。*

## 核心推论 (Key Axioms for Alignment)

1. **情绪是计算结果：** 情绪不是一种原始的生理事实，而是认知系统为了降维处理复杂位势差而生成的一种“用户界面”或“故事”。
2. **焦虑即裸露：** 焦虑产生于叙事无法覆盖位势差（PG）的时刻，即“故事编不下去了”，结构性矛盾直接暴露在意识中。
3. **沟通即对齐叙事：** 有效的沟通不是交换信息，而是对齐双方的“叙事压缩算法”，确保双方处理 PG 的方式兼容。
